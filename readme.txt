Nodejs Application(Only Backend)
System Requirement Node
First extract the files from the zip folder
After extracting go the directory CodingAssignment
Open Terminal
type the command:node index.js
After you find a print statement in terminal says that Server is up on the port 8081
And than open any browser and type
1)To list all the Products
 Type : https://localhost:8081/products
2)To search a Product by its name
 Type: https://localhost:8081/search?name=RedBull
3)To retrive the list of favourite Products
 Type: https://localhost:8081/favourites
